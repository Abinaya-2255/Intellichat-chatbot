# ⚡ INTELLICHAT: Adaptive AI Concierge 
> **A private, domain-specific AI interface built for high-end specialized business environments.**

## 🌟 The Vision
IntelliChat is designed to move away from generic, "corporate" chatbot aesthetics. It provides a premium **Digital Concierge** experience that prioritizes user engagement and data privacy by running entirely on local hardware.

---

## 🎨 Design & UX Philosophy
The interface is built on **Cyber-Grid principles**, moving beyond flat design to create an immersive, HUD-like environment.

* **Aesthetic:** Neon cyan glows, glassmorphism, and radial background grids.
* **Engagement:** Utilizing motion (video interfaces) and typewriter effects to humanize the AI interaction.
* **User Focus:** Designed for boutique businesses where brand identity and "vibe" are as important as function.

---

## 🚀 Technical Stack
* **Frontend:** HTML5, CSS3 (Custom Grid System), JavaScript (Typewriter Logic).
* **Backend:** Flask (Python).
* **Intelligence:** Ollama (Phi-3) — *100% Local and Private.*

---

## 🛠️ Installation & Setup

1.  **Install Ollama:** Download from [ollama.com](https://ollama.com).
2.  **Pull the Model:** ```bash
    ollama run phi3
    ```
3.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/Abinaya-2255/Intellichat-chatbot.git](https://github.com/Abinaya-2255/Intellichat-chatbot.git)
    ```
4.  **Install Dependencies:**
    ```bash
    pip install flask requests
    ```
5.  **Launch:**
    ```bash
    python app.py
    ```
