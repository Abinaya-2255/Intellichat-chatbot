⚡ INTELLICHAT: Adaptive AI Concierge
A highly futuristic, domain-specific AI interface built for specialized business environments.

## 🚀 The Tech Stack
- **Frontend:** HTML5, CSS3 (Cyber-Grid/Glassmorphism), JavaScript (Typewriter Engine)
- **Backend:** Flask (Python)
- **Intelligence:** Ollama (Phi-3) running locally for maximum privacy.

## 🎨 Design Philosophy
The UI was built on **Cyber-Grid principles**, utilizing high-contrast neon cyan glows and radial grids to create an immersive, HUD-like experience that stands out from generic white-label chatbots.

## 🛠️ Installation
1. Install [Ollama](https://ollama.com) and run `ollama run phi3`.
2. Clone this repo: `git clone ...`
3. Install dependencies: `pip install flask requests`
4. Run: `python app.py`
